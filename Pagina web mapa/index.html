    About this resource...

    Isaac Buzo Sánchez propone este mapa físico de Europa a los alumnos del primer ciclo de ESO para que practiquen sus conocimientos sobre las cordilleras y macizos, ríos, mares, penínsulas, llanuras y depresiones, cabos, estrechos, canales y golfos. Una vez estudiado el mapa con todos los elementos identificados, se puede descargar el mismo mapa pero mudo para comprobar lo que se ha aprendido.

    Para obtener más recursos educativos consulta el índice y el buscador facetado de Didactalia.
